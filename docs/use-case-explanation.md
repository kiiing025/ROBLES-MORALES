# WeatherHub Use Case Notes

## Roles

### Guest
- View Landing Page
- Register Account
- Login

### User
- Login
- Logout
- View Dashboard
- Search Weather
- Save Location
- View Search History
- View Preferences

### Admin (future-ready role)
- Read Users
- Update Users
- Delete Users
- Monitor User Activity

## Why use case diagrams matter
Use case diagrams define who can access a feature and what each role is allowed to do. They help set clear system boundaries, reduce feature confusion, and prevent development mistakes such as giving a guest access to a protected page.
